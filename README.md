# 游戏公司对账系统

一个现代化的内部对账管理系统，用于记录和管理游戏公司的财务流水。

## 功能特性

- ✅ 收入/支出记录管理
- ✅ 数据分类统计
- ✅ 实时余额计算
- ✅ 数据搜索和筛选
- ✅ CSV 数据导出
- ✅ 响应式设计，支持深色模式
- ✅ 本地数据存储

## 技术栈

- **Next.js 14** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **date-fns** - 日期处理

## 本地开发

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 在浏览器中打开 [http://localhost:3000](http://localhost:3000)

## 部署到 Vercel

1. 将代码推送到 GitHub 仓库
2. 在 [Vercel](https://vercel.com) 中导入项目
3. Vercel 会自动检测 Next.js 项目并完成部署

或者使用 Vercel CLI：
```bash
npm i -g vercel
vercel
```

## 使用说明

1. **新增记录**：点击"新增记录"按钮，填写日期、类型、类别、金额和描述
2. **编辑记录**：在记录列表中点击"编辑"按钮
3. **删除记录**：在记录列表中点击"删除"按钮
4. **搜索**：使用搜索框按描述或类别搜索
5. **筛选**：使用下拉菜单按类型筛选（全部/收入/支出）
6. **导出**：点击"导出 CSV"按钮将数据导出为 CSV 文件

## 数据存储

数据存储在浏览器的 localStorage 中，刷新页面不会丢失数据。

## 注意事项

- 这是内部使用的系统，数据存储在本地浏览器中
- 如需多设备同步，建议使用云数据库或后端服务
- 清空数据操作不可恢复，请谨慎操作

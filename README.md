# 游戏公司对账系统

一个现代化的内部对账管理系统，用于记录和管理游戏公司的财务流水。

## 功能特性

### 核心功能
- ✅ 收入/支出记录管理（新增、编辑、删除）
- ✅ 数据分类统计和实时余额计算
- ✅ 数据搜索和多维度筛选（类型、日期范围）
- ✅ CSV 数据导出和导入
- ✅ JSON 数据备份和恢复
- ✅ 打印功能

### 统计分析
- ✅ 类别统计图表（可视化展示各类别收支情况）
- ✅ 月度统计报表（按月份汇总收支数据）
- ✅ 日期范围筛选（支持自定义时间段查询）
- ✅ 快速筛选（本月、全部、收入、支出）

### 界面特性
- ✅ 响应式设计，完美适配各种设备
- ✅ 深色模式支持
- ✅ 标签页切换（记录列表、类别统计、月度统计）
- ✅ 现代化 UI 设计

## 技术栈

- **Next.js 14** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **date-fns** - 日期处理

## 本地开发

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 在浏览器中打开 [http://localhost:3000](http://localhost:3000)

## 部署到 Vercel

1. 将代码推送到 GitHub 仓库
2. 在 [Vercel](https://vercel.com) 中导入项目
3. Vercel 会自动检测 Next.js 项目并完成部署

或者使用 Vercel CLI：
```bash
npm i -g vercel
vercel
```

## 使用说明

### 基本操作
1. **新增记录**：点击"新增记录"按钮，填写日期、类型、类别、金额和描述
2. **编辑记录**：在记录列表中点击"编辑"按钮
3. **删除记录**：在记录列表中点击"删除"按钮
4. **搜索**：使用搜索框按描述或类别搜索
5. **筛选**：使用下拉菜单按类型筛选（全部/收入/支出）

### 日期筛选
- 设置开始日期和结束日期进行时间段筛选
- 点击"本月"按钮快速筛选当月数据
- 点击"清除"按钮重置日期筛选

### 数据管理
- **导出 CSV**：将当前筛选的数据导出为 CSV 文件
- **导入 CSV**：从 CSV 文件导入数据（格式：日期,类型,类别,金额,描述）
- **备份数据**：将全部数据导出为 JSON 文件进行备份
- **恢复数据**：从 JSON 备份文件恢复数据
- **打印**：打印当前页面内容（自动隐藏操作按钮）

### 统计分析
- **记录列表**：查看所有对账记录，支持排序和筛选
- **类别统计**：查看各分类的收支统计和可视化图表
- **月度统计**：查看按月份汇总的收支报表

## 数据存储

数据存储在浏览器的 localStorage 中，刷新页面不会丢失数据。

## 注意事项

- 这是内部使用的系统，数据存储在本地浏览器中
- 如需多设备同步，建议使用云数据库或后端服务
- 清空数据操作不可恢复，请谨慎操作

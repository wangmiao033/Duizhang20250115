# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [1.2.0] - 2025-01-15

### 新增功能
- ✨ 添加 Toast 提示系统，所有操作都有友好的成功/错误提示
- ✨ 添加确认对话框组件，重要操作需要确认防止误操作
- ✨ 添加批量操作功能，支持多选和批量删除记录
- ✨ 添加数据排序功能，支持按日期、金额、类别排序（升序/降序）
- ✨ 添加快捷添加功能，快速添加常用类别的记录
- ✨ 添加年度统计视图，按年份汇总收支数据
- ✨ 添加详细统计卡片，显示记录数、平均金额、最大金额等多维度统计
- ✨ 添加 Excel 导出功能，支持导出为 .xls 格式

### 优化改进
- 🎨 优化界面布局，新增年度统计和详细统计标签页
- 🎨 改进批量操作栏显示效果
- 🎨 添加排序控件，方便切换排序方式
- 🎨 优化动画效果，提升用户体验

### 用户体验
- 💡 所有操作都有明确的反馈提示
- 💡 重要操作需要确认，防止误操作
- 💡 批量处理功能提高操作效率
- 💡 快捷操作简化常用功能的使用

## 回退说明

如需回退到之前的版本，请按以下步骤操作：

1. **查看更新日志**：在 CHANGELOG.md 中查看各版本的变更内容
2. **备份当前数据**：使用"备份数据"功能导出当前数据
3. **恢复数据**：如果需要恢复之前的数据，使用"恢复数据"功能导入备份文件
4. **代码回退**：如需回退代码版本，使用 Git 命令：
   ```bash
   git log --oneline  # 查看提交历史
   git checkout <commit-hash>  # 回退到指定版本
   ```
